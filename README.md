# react-native-awesome-mqtt

MQTT Client for React Native

## Installation

```sh
npm install react-native-awesome-mqtt
```

## Usage

```js
import AwesomeMqtt from "react-native-awesome-mqtt";

// ...

const result = await AwesomeMqtt.multiply(3, 7);
```
